League AI Coach — LLM tool-using agent & trace-grounded evaluation

TypeScript
LLM agentsevaluationExpressSupabaseZod

A 21-tool streaming LLM coaching agent over League of Legends match data, persisting every tool call, argument, and result per turn. I built a trace-grounded evaluation harness (deterministic checks + a hand-authored gold-match dataset + LLM-as-judge), and derived a 6-class failure taxonomy that guided successive mitigation rounds — raising grounding from 46 to 83 (+37). A controlled ablation then isolated pre-injection's causal effect as −31% tool calls. in progress: multi-agent match analysis

Evaluation framework & project summary available on request.

Implicit Negation in Sentiment Classification

COMP 550 (NLP) · Python
NLP researchBERTPyTorchHuggingFace

An ACL-format research paper (team of 3; my role: model implementation & experiments) studying how traditional models (Naive Bayes, Logistic Regression) vs. fine-tuned BERT handle implicit negation on IMDb — evaluation stratified by negation type, plus a lexicon-augmentation intervention that recovers ~72% of Logistic Regression's degradation (on a small N=55 subset; indicative) but cannot close the gap to BERT.
